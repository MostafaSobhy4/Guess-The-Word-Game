# Word Guessing Game

A simple word guessing game (like Wordle) built with HTML, CSS, and JavaScript.
The player tries to guess the hidden word within a limited number of tries and he has a limited number of hints.

# Features

- User-friendly input system (auto-focus and skip filled inputs).

- Modern UI with disabled/enabled rows per try.

- Smooth navigation between letters (auto uppercase, skip filled boxes).

- Win/Lose messages with the correct word revealed.

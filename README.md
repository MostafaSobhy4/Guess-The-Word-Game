# Word Guessing Game

A simple word guessing game (similar to Wordle) built with HTML, CSS, and JavaScript.
The player tries to guess the hidden word within a limited number of tries and has a limited number of hints.

# Features

- User-friendly input system – Auto-focus and skip filled inputs for smooth typing.
- Modern UI – Rows are enabled/disabled per try for better game flow.
- Smart navigation – Auto-uppercase letters and skip filled boxes.
- Win/Lose messages – Displays the correct word when the game ends.
- Hint system – Limited hints to assist players during the game.
- Responsive design – Works well on desktop and mobile devices.

# Built With

- HTML
- CSS
- JavaScript
